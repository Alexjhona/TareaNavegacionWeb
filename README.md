## Ejercicio 6: Historial de Navegación Web

Este proyecto implementa una simulación de un navegador web en Java utilizando un ArrayList para gestionar el historial de páginas.

Se utiliza un índice para controlar la posición actual y permitir navegar entre páginas.

Funcionalidades:
- Visitar página
- Ir atrás
- Ir adelante
- Mostrar historial
- Buscar en historial
- Limpiar historial

Al visitar una nueva página, se eliminan las páginas posteriores a la posición actual para simular el comportamiento real de un navegador.

Se aplicaron mejoras de calidad de código utilizando SonarQube, reemplazando salidas por consola con logger y optimizando expresiones.
